# 📨 Telegram Рассылка PRO v16.0

Бот для управления массовой рассылкой сообщений в Telegram через Telethon-аккаунты.  
Управление осуществляется через Telegram-бота (aiogram).

---

## ⚙️ Установка

```bash
pip install telethon aiogram
python spam.py
```

---

## 📁 Структура файлов

Все данные хранятся в папке `bot_data/`, которая создаётся автоматически при запуске.

| Файл | Описание |
|---|---|
| `accounts.json` | Список Telegram-аккаунтов (номер, api_id, api_hash, сессия) |
| `targets.json` | Чаты/каналы/боты для основной рассылки |
| `messages.json` | Тексты сообщений для рассылки |
| `settings.json` | Глобальные настройки (интервал, задержка, расписание) |
| `stats.json` | Статистика основной рассылки |
| `dm_settings.json` | Настройки модуля «Активные → ЛС» |
| `dm_stats.json` | Статистика ЛС-рассылки |
| `dm_sent.json` | База обработанных пользователей (защита от повторов) |
| `monitor.json` | Настройки модуля живого мониторинга |
| `mass_dm.json` | Настройки модуля массовой рассылки по базе |
| `mass_users.txt` | Загруженный список пользователей для массовой рассылки |
| `notify_users.json` | Список операторов, получающих уведомления |
| `log.txt` | Лог всех операций |

## 🧩 Модули

### 1. Основная рассылка
Отправляет сообщения в выбранные чаты/каналы по расписанию.

- Добавляй чаты через **📋 Мои чаты → 🔍 Сканировать**
- Включай/выключай каждый чат отдельно
- Настраивай отдельное сообщение для каждого чата или одно общее
- Поддержка медиафайлов (фото, документы)
- Ротация сообщений, перемешивание, расписание по часам

---

### 2. 💌 Активные → ЛС
Анализирует активность в указанных чатах и пишет в личку самым активным участникам.

- Сканирует N последних сообщений в чате
- Берёт топ-N самых активных участников
- Пишет только тем, кому ещё не писали (общая база `dm_sent.json`)
- Пропускает ботов, удалённые аккаунты, закрытые ЛС

**Настройки:**
- `✏️ Текст для ЛС` — текст сообщения
- `📌 Добавить чат` — ID, @username или ссылка на чат
- `🔢 Кол-во топ` — сколько участников брать (1–200)
- `📈 Лимит сканирования` — сколько сообщений анализировать (10–5000)
- `👁 Предпросмотр топ` — посмотреть список без отправки
- `📂 База отправленных` — просмотр обработанных
- `♻️ Сбросить базу` — начать заново

---

### 3. 👁 Мониторинг в реальном времени
Следит за сообщениями в чатах и мгновенно пишет в ЛС каждому кто написал.

- Работает через Telethon event handler
- Пишет сразу после появления нового сообщения
- Использует общую базу `dm_sent.json` чтобы не писать повторно
- Автовосстановление при перезапуске бота

**Настройки:**
- `✏️ Текст (мон.)` — отдельный текст для мониторинга (или общий из модуля 2)
- `📌 Добавить чат (мон.)` — чат для отслеживания
- `🔗 Использовать общую базу` — вкл/выкл проверку повторов

---

### 4. 📤 Массовая рассылка по базе
Отправляет сообщения заранее подготовленному списку пользователей.

- Загрузка через `.txt` файл или текстом в чат
- Поддерживаемые форматы: `@username`, `username`, `123456789`, `https://t.me/username`
- Строки начинающиеся с `#` — комментарии, игнорируются
- Прогресс отчёт каждые 10 пользователей
- Интеграция с общей базой `dm_sent.json`

## 👤 Аккаунты

Добавление через бот: `👤 Аккаунты → ➕ Добавить`

Формат ввода:
```
+79991234567 api_id api_hash
```

Код авторизации вводить **с пробелами вручную** (`1 2 3 4 5`), не копировать.  
Сессии сохраняются в `bot_data/s_НОМЕР.session`.

Получить `api_id` и `api_hash`: [my.telegram.org](https://my.telegram.org)

## ⚙️ Настройки

| Параметр | Описание |
|---|---|
| `interval_sec` | Интервал между рассылками (секунды) |
| `delay` | Задержка между отправками в одном цикле |
| `shuffle` | Перемешивать порядок чатов |
| `emoji` | Добавлять случайный эмодзи в начало сообщения |
| `auto_del` | Автоудаление отправленных сообщений через N секунд |
| `schedule` | Часы активности (например: 9, 12, 18) — вне часов бот не шлёт |

## 📊 База обработанных (`dm_sent.json`)

Общая для всех трёх ЛС-модулей. Каждая запись содержит:

```json
{
  "user_id": {
    "name": "Имя",
    "username": "username",
    "date": "19.02 21:26",
    "ok": true,
    "reason": "",
    "src": "mass"
  }
}
```

Поле `src`: `mass` — массовая рассылка, `monitor` — мониторинг, отсутствует — модуль «Активные → ЛС».

Сброс базы через бот: `💌 Активные → ЛС → ♻️ Сбросить базу`
